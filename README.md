# UCSB Map

## View on Heroku
https://ucsbmap.herokuapp.com/

## How To Run Locally

* mvn clean package
* java -jar target/ucsbmap-0.0.1-SNAPSHOT.jar

Open new terminal tab

* npm run start

Navigate to localhost:3000 to view
